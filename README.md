# esdm-tracker
## Projected created on MERN Stack (MongoDB+Expressjs+Reactjs+Nodejs)
**Steps to get app running on local machine:**
	1. git clone https://github.com/adrielarce/esdm-tracker.git
	2. cd ./esdm-tracker
	3. npm install
	4. cd ./client
	5. npm install
	6. cd ../server
	7. npm install
	8. cd ..
	9. npm start
